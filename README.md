# рЯ¶Њ JARVIS - Artificial Intelligence Voice Assistant

**JARVIS** is an intelligent voice assistant written in Python, based on Google Gemini AI and Edge-TTS technologies. The project was inspired by the legendary assistant Jarvis character from the movie "Iron Man".

---

## рЯЪА Features

- рЯОЩпЄП **Voice dialogue:** Recognizes user speech and responds in Uzbek.
- рЯІ† **Intelligent AI (Gemini):** Answers any question logically and friendly.
- рЯФК **Natural voice:** Speaks in a high-quality male voice using Microsoft Edge TTS (`SardorNeural`).
- рЯЦ•пЄП **System management:** - Open/close applications such as Telegram, Excel, Word and Calculator.
- Search for information on Google.
- Search YouTube and Wikipedia.
- Tell the weather and current time.
- рЯО≠ **Character:** A professional assistant style that addresses the user as "Mr." or "Sir."

## рЯЫ†пЄП Technologies

- **Text-to-speech:** `Edge-TTS` (Microsoft Azure Neural Voices)
- **Audio Control:** `Pygame`
- **System Handling:** `os`, `webbrowser`, `subprocess`
